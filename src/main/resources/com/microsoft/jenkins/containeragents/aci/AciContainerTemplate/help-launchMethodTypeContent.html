Choose SSH or JNLP to launch your slave.

SSH (RECOMMEND)
Make sure container initialize with a SSH server. Take image jenkins/ssh-slave as a reference

If using SSH, here is a sample of arguments:
 Docker image: jenkins/ssh-slave
 Command: setup-sshd
 Environment Variable: Key: JENKINS_SLAVE_SSH_PUBKEY, Value: Your public key
 Launch Method: SSH

JNLP
Make sure container can access Jenkins master. Take image jenkins/jnlp-slave as a reference
